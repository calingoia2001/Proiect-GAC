1.Descriere Proiect

 

Aplicația este realizată cu ajutorul platformei MATLAB, și este realizată n trei fișiere, unul principal și doua pentru funcții.

 

In prima parte a proiectului avem reprezentata schema unui Tranzistor Bipolar de tip npn n conexiunea Baza Comuna polarizat normal. Cu ajutorul teoremelor lui Kirchkoff sunt calculate curentii si tensiunile tranzistorului. Apoi, pe baza rezultatelor obtinute sunt afisate caracteristicile de intrare si iesire a tranzistorului.

 

In a doua parte a proiectului avem reprezentat un amplificator in configuratie cu baza comuna. Acest amplificator este caracterizat de un castig mare in tensiune, de un castig maxim in curent egal cu 1 si avand o rezistenta de intrare scazuta. Pe baza formulelor, aplicatia afiseaza castigurile si rezistenta de intrare avand niste date de intrare stabilite. De asemenea, sunt afisate semnalul de intrare, semnalul de iesire amplificat si punctul static de functionare.
